\doxysubsection{Detailed Description}
Classe base para o View dos corpos. 

Classe base para o View de um corpo, possui a posicao, o angulo, O source da imagem da textura, o render que a textura vai se ligar E o S\+D\+L\+\_\+\+Rect para a posicao
